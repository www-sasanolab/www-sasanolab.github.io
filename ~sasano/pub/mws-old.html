Haskell Codes for Maximum Weightsum Problems

  • the maximum independent-sublist sum problem

  • the maximum two disjoint paths problem on series-parallel graphs

  • the party planning problem, the group organizing problem, and the supervisor chaining problem

    Note: The definition of phi2 in the paper is wrong (in Step 3 of Section 5.2 (p.8 and p.9)). The definition of phi2 should be as follows.

    phi2 (a1,b1) (a2,b2) = (not (b1 and b2) and a1 and a2, b2)

    The Haskell code to solve the party planning problem in this file company.hs is written using this function phi2.
    phi2 can be simplified as follows.

    phi2 c0 c0 = c0
    phi2 c0 c1 = c1
    phi2 c0 c2 = c0
    phi2 c0 c3 = c1
    phi2 c1 c0 = c0
    phi2 c1 c1 = c1
    phi2 c1 c2 = c0
    phi2 c1 c3 = c1
    phi2 c2 c0 = c0
    phi2 c2 c1 = c1
    phi2 c2 c2 = c2
    phi2 c2 c3 = c3
    phi2 c3 c0 = c0
    phi2 c3 c1 = c1
    phi2 c3 c2 = c2
    phi2 c3 c3 = c1

    And the property description p, p' of the supervisor chaining problem is wrong (in Section 6.1 (p.9)). These should be as follows.

    p (Leader v []) = True
p (Leader v (t:ts)) =
    if marked v then
       if nm t then p (Leader v ts)
       else marked (getLeader t) and p t and
            ol (Leader v ts)
       else if nm t then p (Leader v ts)
            else p t and nm (Leader v ts)

ol (Leader v []) = marked v
ol (Leader v (t:ts)) = nm t and ol (Leader v ts)

p' (Root v) = True
p' (Join t1 t2) = if lm' t2 then
                     if nm' t1 then p' t2
                     else lm' t1 and p' t1 and ol' t2
                  else if nm' t1 then p' t2
                       else p' t1 and nm' t2

ol' (Root v) = marked v
ol' (Join t1 t2) = nm' t1 and ol' t2
    The properties ol and ol' means only leader (root) marked.
    The Haskell code to solve the supervisor chaining problem in this file company.hs is obtained by tupling these property descriptions p', lm', nm', ol'.

  • the maximum segment sum problem, the maximum even number segment sum problem, and the maximum even length segment sum problem

    Note: In the paper, the definition of evens in the case the argument is singleton is wrong (in Section 6.3 (p.10)). It should be as follows.

    evens [x] = if marked x then even (weight x) else True

    The Haskell code in this file mss.hs is obtained by using this definition.
