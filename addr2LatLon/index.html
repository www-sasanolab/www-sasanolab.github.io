Programs for transforming addresses to latitudes and longitudes

This page provides an implementation of transforming addresses to latitudes and longitudes by using the service ArcGIS provided by ESRI. This is used in the global PBL held in February 18-27, 2019.
  • A program in JavaScript for transforming addresses to latitudes and longitudes by using ArcGIS service provided by ESRI
  • A program in JavaScript for transforming addresses to latitudes and longitudes by using ArcGIS service provided by ESRI with displaying a map by OpenStreetMap with the library Leaflet (updated on 14 Feb 2020 following the update of Leaflet)

How to use

Download either of the above html files and open the file in some browser like Firefox. Then input some address in the box and push the search button.
