Japanese Syntax Analysis

2010/05/11

自然言語処理システム特論の課題で作りました。

  • CYK法による日本語構文解析
    (This script do syntax analysis for Japanese by CYK algorithm which is for natural language parsing)
  • 形態素解析はYahooのテキスト解析APIを利用
    (using Yahoo Web API for morphological analysis)
  • 構文規則を簡単に書き換え可能
    (We can rewrite production rules easily. Production rules has to be written in Chomsky normal form. )
  • 結果をグラフィカルに表示
    (A result of parsing is shown as graph. )
  • デフォの構文規則が弱いので大抵の文は解析に失敗するよ
    (There are many sentences that cannot be parsed since default syntax rules are not enough)

Input

品詞(part of speech):形容詞,形容動詞,感動詞,副詞,連体詞,接続詞,接頭辞,接尾辞,名詞句,動詞,助詞,助動詞,特殊

例：アマゾンで買った分厚い本を売ったことで得たお金で遊んだ

Yahoo形態素解析では「昨日」が副詞にならないため授業の例では上手くいきません。
形態素解析の結果、形態素が1個以下の場合は解析を行いません。この実装の仕様です。
あんまりグラフがでかくなると線が被ります。手抜きです。すいません。

外部ファイルとか(external files and using library)

  • productionParser.js:構文規則の読み込む簡易パーサ(production rule scanner)
  • processing.js: Graphics library for Canvas element of HTML5
  • JKL.ParseXML: XML parsing library
