<h1>Lambda-mode version 0.06</h1>
This page provides an old implementation of variable name completion in Emacs-mode
based on the algorithm presented in <a href="http://www.sic.shibaura-it.ac.jp/~sasano/pub/ppl2010GotoSasano.pdf">our paper (in Japanese)</a>.
This version is very slow. We strongly recommend to use the current version.
